Il sito web è stato realizzato prendendo ispirazione dall'esercizio XHTML-9
"Città del fiore di zucca" unitamente all'esercizio CSS-3.
Il layout della pagina iniziale (index.html) è organizzato mediante l'utilizzo di tabelle.
La pagina index.html rappresenta la pagina introduttiva del sito e i link presenti (nel menu) permettono la navigazione.


Caratteristiche di XHTML e di CSS implementate

- Dichiarazione dtd (strict.dtd)
- prologo XML
- Dichiarazione namespace
- Utilizzo dei principali elementi XHTML: <p>, <h1>, <ul>, <a>, <img>, <table>, <hr />, ...
- Elementi block-level / in-line
- Utilizzo di CSS inline ed external + selettori CSS
- specifica colori #XXXXXX
- positioning: position, top, left, float, ...


Infine, la validità dei documenti è stata controllata tramite i validatori:
- http://validator.w3.org/ (validatore di markup)
- https://jigsaw.w3.org/css-validator/ (validatore CSS)
